# Salvation
Игра на Godot 4.6.1

Для игроков (скачать готовую сборку)
1. Перейдите на вкладку "Actions"
2. Выберите последний успешный запуск (зелёная галочка)
3. Внизу страницы скачайте "Artifacts" под вашу платформу:
   - `game-windows.zip` — для Windows
   - `game-linux.zip` — для Linux
   - `game-macos.zip` — для macOS
4. Распакуйте и запустите игру

Для разработчиков (начать работу)
1. Установите Godot 4.6.1 (обязательно эту версию!)
2. Установите GitHub Desktop
3. Склонируйте репозиторий:
   - Откройте GitHub Desktop → File → Clone repository → URL
   - Вставьте: `https://github.com/letishka/Salvation`
4. Откройте проект в Godot (Import → выбрать папку → `project.godot`)

Технологии
- Godot 4.6.1
- Git + GitHub
- GitHub Actions (автоматическая сборка)
